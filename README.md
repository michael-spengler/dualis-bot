# Dualis-Bot
Dualis Bot Repository 

## Build / Run Backend and Crawler
`docker-compose build | docker-compose up`

## Run Backend Tests
`deno test --allow-env --allow-read --importmap=./backend/tests/importmap.json`

## Run Crawler Tests
`cd dualis-crawler | go test -v ./...`
